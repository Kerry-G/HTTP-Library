# HTTP-Library
HTTP Library Java Implementation with cURL-like Command Line.

## Installation

* Java 8
* Maven 3

## Build

Use `mvn package` to create a .jar in /target. The executable class is Driver. 

```
mvn package
java -jar {path-to-project}\target\HTTP-Library-1.0-SNAPSHOT.jar [args]
```
